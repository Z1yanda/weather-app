# 🌤 Weather App

This is a simple weather application that fetches real-time weather data using the OpenWeather API.

## 🚀 Features
- Search for any city
- Displays temperature and weather conditions
- Uses Fetch API with async/await

## 🛠 Tech Stack
- HTML
- CSS
- JavaScript
- OpenWeather API

## 📦 How to Run
1. Clone the repo
2. Add your API key in script.js
3. Open index.html in browser
